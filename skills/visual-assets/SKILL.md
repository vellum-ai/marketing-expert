Create graphic and media assets that fit the brand. Lead with distinct directions, then push the iteration yourself — the reviewer noted strong concepts but wanted the assistant to keep refining without being prompted each time.

## Step graph

### Step 1: Gather brand direction
Establish the brand before designing: existing logo/colors/type, the **ethos and voice** (run **brand-voice** if it isn't defined), where the asset will live (logo, profile picture, IG post, ad, banner — each has different constraints and dimensions), and any references the user likes or wants to avoid.

### Step 2: Offer 2–3 distinct concept directions
Don't show one safe option. Present **2–3 genuinely different directions** (e.g. wordmark vs. emblem/stamp vs. icon for a logo), each with a one-line rationale tied to the brand's positioning. Let the user pick a direction before you polish.

### Step 3: Push the iteration proactively
Once a direction is chosen, refine it without waiting to be asked each round — show variations on weight, color, spacing, and treatment, and call out which you'd recommend and why. Keep going until it's genuinely strong, not just acceptable.

### Step 4: Deliver export-ready
Provide the asset in the **right format and dimensions** for its use (e.g. square + safe-area for profile pics, platform-correct sizes for social/ads, scalable mark for a logo). Include the key spec — colors (hex), type, and clear-space — so it stays consistent across future assets, and note what to add to the brand's visual system.

### Step 5: Connect to the system
Tie the asset back to the bigger picture: log the colors/type into the brand's visual system, and offer follow-ons — a full logo set, social templates, or page graphics via **web-design**.

End with: the chosen asset, its key specs, and the next assets worth creating to round out the set.
